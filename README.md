# Memory Match

A simple browser-based memory matching game. Flip cards two at a time to find all 8 matching emoji pairs in as few moves as possible.

## How to play

Open [`index.html`](index.html) in your browser and start clicking cards. The game tracks your move count and elapsed time, and lets you restart at any point.

## Running locally

No build step or dependencies required — just open `index.html` directly in a web browser.
